# Práctica 3.1 - Introducción a Lubuntu

## Ejercicios

Haz una captura de pantalla para cada uno de los ejercicios.

1. Abre el menú de aplicaciones y busca el editor de texto. Cuando lo encuentres ejecútalo.
2. Abre una nueva instancia del editor de texto utilizando el lanzador.
3. Busca la aplicación Terminal y añade un icono al lanzador.
4. Abre dos ventanas en cada área de trabajo.
5. Mediante atajos de teclado haz lo siguiente:
    - Cambia la ventana activa.
    - Abre la última aplicación del lanzador.
    - Cambia entre aplicaciones de todas las áreas de trabajo.
    - Cambia el área de trabajo.
    - Minimiza todas las ventanas.
    - Maximiza la ventana actual.
6. Maximiza una ventana arrastándola al borde superior.
7. Abre un terminal y realiza lo siguiente:
    - Abre dos pestañas.
    - Cambia el color de fondo a azul y el color de primer plano a blanco.
    - Aumenta el tamaño de la fuente a 13 puntos.
8. En el terminal, muestra la página de manual del comando ``cp``.
9. Muestra una lista de comandos con los comandos relacionados con la palabra ``user``.
10. Consulta la sintaxis del comando ``sh``. Indica si las siguientes ejecuciones del comando se ajustan a la sintaxis o darían error:
    - ``ssh``.
    - ``ssh usuario@servidor``.
    - ``ssh servidor``.
    - ``ssh -C zip servidor``.
11. Ejecuta los siguientes comandos para obtener la siguiente información. Después de la ejecución de cada uno limpia la terminal.
    - Muestra la identificación del usuario y grupos al que pertenece el usuario que ha abierto sesión.
    - Muestra la identificación de usuarios y grupos del usuario ``root``.
    - Muestra el directorio activo.
    - Lista el histórico de comandos.
    - Muestra la información de todas las interfaces de red.
    - Muestra la información de la interfaz de red cableada.
    - Muestra la fecha del sistema.
    - Muestra el calendario del mes de diciembre del año actual.
    - Muestra la versión del kernel.
    - Muestra el nombre del sistema operativo.
    - Muestra el nombre del equipo.

## Entrega

Crea un documento PDF, a partir de un Word o Google Docs, con las siguientes consideraciones:

- El nombre del archivo a entregar en la plataforma Moodle Centros debe tener el siguiente formato: `Apellido1Apellido2_Nombre_SOM_UD3_P1.pdf`.
- Portada con imagen, nombre del alumno, título y curso.
- Encabezado con nombre de la práctica y materia.
- Pie de página con nombre, número de página insertado, curso, año escolar, instituto.
- Índice o tabla de contenidos generada automáticamente.
- Tipo de letra: Times New Roman 12 o similar, interlineado sencillo, espaciado anterior 6p.
- Uso correcto de la ortografía.
- Texto justificado con uso de salto de páginas correcto.
- Enlaces web de referencia cuando se coge información de un sitio web.
- Las fotos que se puedan ver correctamente al ampliarlas y que no ocupen mucho espacio.
- Insertar Título de foto que haga referencia al apartado y ejercicio correspondiente.
- Para las capturas de pantalla de las máquinas virtuales:
    - Se captura todo el escritorio de la máquina virtual donde se pueda leer el nombre de la máquina, con el fondo de su plataforma Moodle, donde se pueda ver imagen de perfil. Toda aquella práctica que no cumpla este requisito no será evaluada.
    - Al igual que en los Títulos de las fotos, en el Título de la captura de pantalla irá referenciado el apartado y ejercicio al que corresponde.
